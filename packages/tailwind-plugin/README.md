<h3>@junipero/tailwind-plugin</h3>
<p>A plugin to use Junipero styles with Tailwind CSS</p>

</div>

## Installation

```bash
yarn add @junipero/tailwind-plugin
```

## Usage

```javascript
// tailwind.config.js
const junipero = require('@junipero/tailwind-plugin');

module.exports = {
  plugins: [
    junipero,
  ],
};
```
